# interactive-3d-product-showcase
An Apple-inspired high-performance 3D product marketing page built with React, TypeScript, and Vanilla Three.js. Features GSAP scroll-driven animations, custom GLSL shader effects, and advanced Draco model compression for a cinematic WebGL experience.
